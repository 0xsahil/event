# Event Project Starter

* Clone the project:
```
npx degit 0xsahil/event
```
* Navigate to the project directory and install dependencies:
```
cd Event && npm install
```
* Start the project:
```
npm run dev
```
The app should now be available locally at http://localhost:5173/
